### Prerequisites

### Setup Instructions

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/sentosa-app.git
   cd sentosa-app
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   ```

4. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000)
